Uma API RESTful desenvolvida com Node.js, Express e MongoDB para simular o funcionamento de uma livraria.

## Tecnologias utilizadas
- Node.js
- Express
- MongoDB
- Mongoose
- Dotenv

## Funcionalidades
- Cadastro de livros
- Listagem de livros
- Atualização de informações dos livros
- Remoção de livros
- Cadastro de autores
- Relacionamento entre livros e autores
